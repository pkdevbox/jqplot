seriesDefaults

Default properties which will be applied directly to the series.

Properties

Properties / base - the logarithmic base, commonly 2, 10 or Math.E tickDistribution - ‘even’ or ‘power’.  ‘even’ gives equal pixel spacing of the ticks on the plot.  ‘power’ gives ticks in powers of 10.
